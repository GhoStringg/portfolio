02 EMBEDDED SYSTEMS

4-Channel Relay Module

An essential component for controlling high-power devices with low-power microcontroller signals. This 4-channel relay module allows for independent control of up to four devices, making it ideal for home automation, industrial control, and IoT projects.

Power Supply Unit

A reliable power supply is the backbone of any electronic system. This custom-built power supply unit delivers stable voltage and current, ensuring that all connected components operate within their specified parameters for optimal performance and longevity.

Air Quality Monitoring

An environmental monitoring system that tracks air quality parameters including particulate matter, volatile organic compounds, and gas concentrations. This project demonstrates the integration of multiple sensors and data visualization for real-time environmental awareness.

03 WEBSITES

BrainBout

A reviewer website for DHVSU (now known as Pampanga State University) students. Features include modules, quizzes, login system, scores tracking, and a leaderboard to foster healthy academic competition among students.

MUSIKA

A Node.js server application that integrates with the Spotify and YouTube APIs. This music search website allows users to discover and explore music across both platforms, providing a unified search experience for music enthusiasts.

Eleganza

An e-commerce platform dedicated to elegant clothing and fashion. Features include product catalog, shopping cart, user authentication, order management, and a seamless checkout experience for online shoppers.
